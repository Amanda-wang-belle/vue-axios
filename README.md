# vue-axios
项目介绍：axios在vue-cli中的封装。
##新建项目
```
vue create vue-axios（项目名称不可出现大写）
```
##安装Axios
```
npm run axios
yarn add axios
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### 在vue组件中引入axios
```
import axios from 'axios'
```
